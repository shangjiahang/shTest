个人教学项目

项目时间 : 2016 - 2017

项目名称 : Java五子棋小游戏设计/推箱子小游戏设计

完成奖励 : 优秀学员

项目介绍

五子棋游戏开发： 五子棋的游戏开发是一个控制台开发所实现的小项目通过用┳、┻、┣、┫┏、┓、┗、┛、╋ 这些符号来显示整个棋盘的，用数组创建棋盘，将上述符号利用坐标循环定义到数组中从而显示出棋盘的形状；●、○使用上述两个符号作为对战双方，使用while循环控制双方每一步的落子，在每一次落子时都会调用判断函数来判断在落子的坐标上是否有已经落得棋子，并判断落子后该棋子的八个方向是否形成五子连珠最后通过三个if判断棋局胜、负或平局。

推箱子游戏开发：推箱子的游戏通过创建窗体并定义一个JLabel数组来标记整个地图，数组中用0、1表示地图中的空地和障碍物；使用JLabel制作背景、人物、目标、目的地，并通过设置图片组件的位置、大小将图片加载到窗体中（在加载图片时要注意加载的先后顺序，地图必须最先加载才能使得人物、目标、目的地显示在地图上面）；给人物、目标、目的地的坐标分别设定值，通过ASCLL码设定移动键来选择人物的移动方向，并在移动时做出能否推动目标进行移动的判断来更改人物及目标的坐标值；通过叠加目标与目的地的值来判断是否将目标移动到目的地。 我在项目中使用switch选择不同的数组从而实现地图的更新和判断玩家进行到哪一关卡，并在switch外嵌套while循环使得玩家能够进行不同关卡的选择。

项目时间 : 2018.02 - 2018.04

项目名称 : Java OA系统设计(BBS论坛设计/新闻发布系统设计)

项目描述 :

BBS论坛项目分别实现了以下功能：

  • 查看主版面中的根贴（将数据库中的所有根贴以分页的形式进行展示）；
  • 查看根帖详细内容（通过点击根帖或回复的帖子可以查看详细信息）；
  • 管理员（用户）登录（通过设计数据库表给予不同的权限从而实现不同的使用不同的功能）；
  • 管理员（用户）注册；
  • 管理员（用户）注销；
  • 管理员（用户）权限实现（包括管理员删除帖子，用户回复帖子）；

论坛项目运用到的技术： JDBC(JAVA 数据库连接):通过与数据库建立连接，并发送操作数据库的语句，对数据源进行访问、操作；本项目在登录验证、帖子内容显示、回帖内容录入等的页面中都是使用JDBC技术实现web应用程序与数据库的之间的对话。 Javascript: 本项目在回帖的功能实现中运用了Javascript技术，通过Javascript编写回帖验证删除空格函数，使的整个系统的健壮性有更好的提高。 html ：本项目的整体界面设计通过html编写实现网页的设计。 jsp （Java服务器页面）：通过在html实现的网页中插入Java程序段，从而实现通过网页表单获取用户输入数据、访问数据库然后动态的创建网页。 本项目中用到的服务器为 tomcat 7.0版本。使用的数据库为 MySQL,本项目中创建有两个表，users用来存放用户账户的数据，article用来存放项目中贴子的文件数据。

项目介绍

  • 使用前端框架 SSM(Spring,SpringMVC,Mybatis)
  • 前端页面设计JSP技术
  • 后端Java语言
